# Dewey Website

Public website for the Dewey book reading tracker iOS app.

## Pages

- `/` - Landing page
- `/privacy/` - Privacy policy

## Deployment

This repo uses GitHub Pages. Any push to `main` automatically deploys to:

**https://dewey.github.io**

## Setup

This repo was created as part of the `dewey` GitHub organization to host public-facing pages for the Dewey iOS app (which is in a private repo).
